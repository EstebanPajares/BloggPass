MIGRAR DE LA RAMA MASTER A MAIN EN GITHUB

Procedimiento para migrar de la rama master a main:

  • Desde nuestro GitHub hacemos una copia de nuestro repositorio, ya sea por HTTPS o SSH. Ejem:
  • Vamos a nuestro repositorio local
  • git clone git@github.com:EstebanPajares/hyperblog.git
  • git status // nos dirá que estamos en la rama master
  • git branch -m master main // Cambiaremos de la rama master a main por defecto
  • git push -u origin main //git push origin main // Enviamos al repositorio remoto
  • git symbolic-ref refs/remotes/origin/HEAD refs/remotes/origin/main // Hacemos que el HEAD apunte a main
  • Actualizamos la rama por defecto a main desde GitHub
  • Settings/Branches
  • Seleccionamos a main por defecto/Update
  • Luego eliminamos la rama master desde git y GitHub
  • git push origin --delete master
